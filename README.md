## ✅ ¿Qué hace TG Gestión?

TG Gestión es un sistema de **ventas, inventario y caja** para uso local. No necesita internet.

Incluye:
- **Ventas (POS):** cobros rápidos en efectivo o fiado.
- **Inventario:** altas, bajas y control de stock.
- **Caja:** resumen del día y total real en cajón.
- **Reportes:** exportación a Excel con un clic.
- **Usuarios:** roles Vendedor, Admin y SuperAdmin.
- **Tema claro/oscuro** y color personalizado.

## 🧭 Guía Rápida de Uso

### 1) Agregar un producto
1. Entra a **Inventario**.
2. Pulsa **+ Nuevo Producto**.
3. Completa nombre, precio y stock.
4. Presiona **Guardar**.

### 2) Hacer una venta
1. Entra a **Ventas**.
2. Busca el producto.
3. Elige cantidad.
4. Presiona **Agregar al Ticket**.
5. Selecciona forma de pago.
6. Presiona **Cobrar**.

### 3) Ver corte de caja
1. Entra a **Registros**.
2. Arriba verás el resumen del día.
3. El **Total en Cajón** es el efectivo real que debe estar.

### 4) Exportar reportes
1. Entra a **Reportes**.
2. Elige el reporte.
3. Presiona **Descargar Excel**.
